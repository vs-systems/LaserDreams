# Laser Dreams - Premium E-commerce Frontend

Este proyecto es la interfaz frontend estática para **Laser Dreams**, distribuidor directo de equipamiento de iluminación profesional y efectos especiales.

## Tecnologías
- HTML5
- CSS3 (Vanilla, Variables CSS, Grid/Flexbox)
- JavaScript (Vanilla DOM manipulation)

## Características
- **Dark Mode Estricto** con detalles geométricos y colores neón (Cyan Láser).
- **Responsive Design** optimizado para escritorio y móviles.
- **Catálogo Dinámico** generado a través de `js/data.js`.
- **Integración con WhatsApp:** Los CTA generan mensajes personalizados con el nombre del equipo consultado.

## Estructura de Archivos
- `index.html`: Layout principal.
- `css/style.css`: Estilos globales y componentes.
- `js/data.js`: Base de datos de productos (traducida al español).
- `js/main.js`: Lógica de inyección y filtrado de productos.
- `img/logo.svg`: Logotipo oficial generado en SVG vectorial puro.
